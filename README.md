## 📦 Installation & Setup
Clone the repo and run locally:
```bash
# Clone repo
git clone https://github.com/Paul-Adek/Blogr-Landing-Page-React-Vite.git
# Navigate
cd Blogr-Landing-Page-React-vite
# Install dependencies
npm install
# Start development server
npm run dev
